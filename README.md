## üöÄ Installation and Usage

Since this is a new extension and not yet published on the Chrome Web Store, you can install it locally using Developer Mode.

### Prerequisites

* Google Chrome web browser.

### Steps to Install Locally

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/abhii-06/chrome-extension01.git](https://github.com/abhii-06/chrome-extension01.git)
    ```
2.  **Navigate to the Extension Folder:**
    ```bash
    cd chrome-extension01
    ```
3.  **Open Chrome Extensions Page:**
    * Open your Chrome browser.
    * Navigate to `chrome://extensions`.
4.  **Enable Developer Mode:**
    * Toggle the **"Developer mode"** switch located in the top right corner of the Extensions page.
5.  **Load the Extension:**
    * Click the **"Load unpacked"** button that appears.
    * Select the **`AI text summarizer`** folder inside the `chrome-extension01` directory you cloned.
6.  **Pin the Extension (Optional but Recommended):**
    * Click the puzzle piece icon (Extensions icon) next to your profile picture in the Chrome toolbar.
    * Find "AI Text Summarizer" and click the **pin icon** next to it so it is always visible.

### How to Use

1.  Navigate to a webpage with a long article or text you want to summarize.
2.  Click the **AI Text Summarizer** icon in your Chrome toolbar.
3.  Click the **"Summarize"** button within the pop-up.
4.  The generated summary will appear in the box below.
